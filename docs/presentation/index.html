Building Text Worlds with Python

Welcome to the exciting world of MUD (Multi-User Dungeon) development! In this presentation, we'll explore how to create interactive, text-based multiplayer games using Python and the Evennia framework.

By the end of this presentation, you'll understand:

  • What MUDs are and why they're a great learning project
  • How to set up Evennia, a Python MUD development framework
  • The basic architecture of an Evennia game
  • How to create your first game elements
  • Where to go next on your MUD development journey

Let's dive into the world of text-based adventures!

Why Build a MUD?

Multi-User Dungeons (MUDs) are text-based multiplayer games that combine storytelling, roleplaying, and game mechanics in a shared virtual world.

Educational Benefits

  • Practical Python Experience: Apply your Python skills to a real project
  • Full-Stack Development: Work with databases, networking, and web interfaces
  • Object-Oriented Programming: Use classes to model a virtual world
  • Collaborative Coding: Work in teams on different game components
  • Creative Expression: Combine coding with storytelling and world-building

Why MUDs are Perfect for Learning

  • Text-based interface means focusing on logic, not graphics
  • Modular design allows incremental development
  • Immediate feedback as you test your creations
  • Scale from simple to complex at your own pace
  • Learn concepts applicable to modern game and web development

Why Evennia?

  • Python-based: Leverages your existing Python knowledge
  • Django foundation: Built on a robust web framework
  • Active community: Helpful forums and documentation
  • Modern architecture: Supports both web and traditional clients
  • Production-ready: Scalable from classroom projects to full games

Compared to alternatives like mud-pi (too limited) or custom Streamlit solutions (requires extensive development), Evennia offers the ideal balance of turnkey setup and extensibility.

Getting Started with Evennia

Let's set up your first Evennia project! Follow these steps to install Evennia and create your initial game environment.

Step 1: Install Evennia

First, create a Python virtual environment (recommended), then install Evennia:

pip install evennia

Step 2: Create a New Game Folder

Initialize a new game directory structure:

evennia --init mygame
cd mygame

Step 3: Initialize the Database

Set up the initial database schema:

evennia migrate

Step 4: Start the Server

Launch your game server (you'll be prompted to create an admin account):

evennia start

Step 5: Connect to Your Game

Your game is now running! Connect using:

  • Web client: http://localhost:4001
  • Telnet: localhost:4000

Log in with the admin account you created when starting the server.

Evennia Architecture

Understanding Evennia's architecture will help you build your game efficiently. Here's an overview of the key components:

Typeclasses

Typeclasses are Python classes that extend database models, allowing you to create custom game objects with persistent data.

  • Characters: Player-controlled entities
  • Rooms: Locations in your game world
  • Objects: Items, weapons, treasures, etc.
  • Exits: Connections between rooms

Commands

Commands process player input and execute game actions. Each command is a Python class with a func method that defines its behavior.

Scripts

Scripts are persistent processes for timed events and background tasks, such as monster spawning, weather changes, or scheduled events.

Server Configuration

Server settings control everything from connection options to game mechanics. The main configuration file is server/conf/settings.py.

Database

All game objects are stored in a database, allowing the game state to persist across server restarts.

Game Areas & Player Types

Our MUD project is structured around four main areas, each appealing to different player types as described by Richard Bartle's player taxonomy.

CREATE (The Forge)

  • Appeals to: Achievers and Explorers
  • Features: Crafting systems, creating items, seeing tangible results
  • Similar to: Minecraft crafting, RPG gear progression

EXPLAIN (The Lore Halls)

  • Appeals to: Socializers and Explorers
  • Features: NPC interactions, dialogue systems, uncovering lore
  • Similar to: Visual novels, dialogue-rich RPGs

CODE (The Arcane Matrix)

  • Appeals to: Achievers and Explorers
  • Features: Mastering logical systems, discovering how things work
  • Similar to: Programming puzzles, logic games

EXPLORE (The Neon Wilderness)

  • Appeals to: Explorers, Killers, and Achievers
  • Features: Mapping territories, tactical combat, finding rewards
  • Similar to: Dungeon crawlers, roguelikes

Key: A = Achievers, S = Socializers, E = Explorers, K = Killers

Basic Commands

Here are some essential commands for developing and managing your Evennia MUD.

Server Management

# Start the Evennia server
evennia start

# Stop the Evennia server
evennia stop

# Restart the Evennia server
evennia restart

# Check server status
evennia status

In-Game Building

# Create a new room
@create hall:rooms.Room
@desc hall = A spacious hall with high ceilings.

# Create exits between rooms
@dig north = kitchen
@desc north = A doorway leading to the kitchen.

# Create an object
@create sword:objects.Object
@desc sword = A shining steel sword.

# Teleport to a room
@tel hall

# Set attributes on objects
@set sword/damage = 10

Basic Player Commands

# Look at your surroundings
look

# Move in a direction
north

# Examine an object
examine sword

# Get an object
get sword

# Drop an object
drop sword
